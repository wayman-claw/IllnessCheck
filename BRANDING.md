# DayTrace Branding & Publishing Notes

## Product Name

**Recommended app name:** DayTrace

Why:
- calm and modern
- not overly medical
- fits daily pattern tracking
- broad enough for future features

## Home Screen Display Name

Set to:
- **DayTrace**

## App Icon Direction

Recommended concept:
- rounded square background
- soft blue to teal gradient
- central symbol: a white checkmark merged with a subtle heart pulse / journal line
- clean, minimal, premium feel

Alternative icon concepts:
1. **Pulse Drop**
   - water drop + pulse line
   - emphasizes health and hydration
2. **Trace Mark**
   - circular trace ring + checkmark
   - emphasizes daily completion and patterns
3. **Quiet Journal**
   - minimal notebook glyph + heart
   - emphasizes reflection

## Suggested Color Direction

- Primary: soft blue
- Secondary: teal
- Accent: warm coral/pink for mood
- Backgrounds: light neutral system backgrounds

## Publishing Preparation

### Must-have before release
- final app icon in all required sizes
- launch/brand polish
- privacy policy
- notification permission explanation
- onboarding / first-run guidance
- export/share flow
- device testing on real iPhone
- App Store screenshots
- app description and keywords
- verify data persistence/migration strategy

### App Store positioning draft
**Track your day in under a minute.**
Log food, drinks, symptoms and overall wellbeing, then spot patterns over time with a calm, simple interface.

### Keywords draft
- health journal
- symptom tracker
- hydration tracker
- daily check-in
- wellness log
- habit tracking

## Technical Notes

To show the correct app name on the iPhone home screen:
- set `INFOPLIST_KEY_CFBundleDisplayName = DayTrace`

To fully rebrand later:
- optionally rename bundle id and project folder structure after UI stabilizes
- keep repo rename separate from product polish to reduce churn
